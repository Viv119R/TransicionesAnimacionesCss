Practica de transiciones y animaciones

Transiciones con css

Permite cambiar suavemente una propiedad de CSS de una estado a otro durante un periodo de tiempo

Propiedades de las transiciones

  • transition-property: Defien qué propiedad cambiar
  • transition-duration: Define la duración del cambio. (s o ms)
  • transition-timing-function: Controla la aceleración o desaceleraciión
    • linerar -> Velocidad constante.
    • ease (por defefcto) -> Comienza lenta, acelera y luego desacelera.
    • ease-in -> Comienza lenta y acelera
    • ease-out -> Comienza rápida y desacelera.
    • ease-in-out -> Comienza y temina lenta
  • transition-delay: Retrasa el inicio de la transición.
