Transiciones con css

Permite cambiar suavemente una propiedad de CSS de una estado a otro durante un periodo de tiempo

Propiedades de las transiciones

  • transition-property: Defien qué propiedad cambiar
  • transition-duration: Define la duración del cambio. (s o ms)
  • transition-timing-function: Controla la aceleración o desaceleraciión
    • linerar -> Velocidad constante.
    • ease (por defefcto) -> Comienza lenta, acelera y luego desacelera.
    • ease-in -> Comienza lenta y acelera
    • ease-out -> Comienza rápida y desacelera.
    • ease-in-out -> Comienza y temina lenta
  • transition-delay: Retrasa el inicio de la transición.

Transición de color de fondo

La transición de color de fondo de CSS permite cambiar suavemente el color de fondo de un elemento cuando se actica un cambio de estado, como un :hover. Esto crea un efecto visual agradable y fluido que mejora la experiencia del usauario al interactuar con la pagina. Por ejemplo. al pasar el cursor sobre un boton, su color de fondo puede cambiar de azul a rojo, atrayendo la atención del usuario son ser abrupto.

Transición de tamaño de fuente

La transicion de tamaño de fuentre facilita el cambio gradual del tamaño del texto de un elemento. Esto es util para destacar elementos interactivos o importantes al aumentar su tamaño cuando se le pasa el cursor sobre ellos. por ejemplo, un párrafo de texto puede aumentar de tamaño de fuente cuando se sitúa el cursor sobre él, destacando su relevancio o indicando que es interactivo.

Transición de opacidad

La transicion de opacidad permite ajustar gradualmete la transparencia de un elemento. Esto es útil para efectos como desvaneciemient de entrada y de salida, o para resaltar elementos al aumentar su opacidad. Por ejemplo, una imagen o un cuadro de texto puede dejar pasar de ser semi-trasparente a completamente opaco al situar el cursor sobre él, atrayendo la atencion del usuario de manera sutil y elegante.
